Projects

Aircraft Accident Injury Level Prediction Using Machine Learning

This project applies machine learning to analyze and predict injury levels in aircraft accidents using the ICAO Air Accidents dataset (2008–2022). After preprocessing and encoding key features such as aircraft, operator, and location, a Linear Regression model was trained to estimate injury severity.

Python • Pandas • NumPy • Scikit-learn • Matplotlib

Email Spam Classification using NLP & Machine Learning

This project focuses on building a machine learning-based email spam classifier. Text data from emails was preprocessed and transformed using techniques such as tokenization, stopword removal, and feature extraction. Both Naive Bayes and Support Vector Machine (SVM) models were trained to classify emails as spam or non-spam. Model performance was evaluated using metrics such as precision and recall to ensure reliable detection of spam emails.

Python • Pandas • NumPy • Scikit-learn
